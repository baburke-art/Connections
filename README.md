# Connections
Welcome to the standard market! This project aims to standardise the connections between parts of products. Standardised connections are fundamental to creative possibilities with products, reducing waste, challenging "throw-away" culture, planned obsolescence, and greater personalisation.

Standarising the connections between the parts of a product creates a template that offers the opportunity to focus on just one part instead of a proprietary product that just ends up in landfill when one part breaks.

An example of a connection type is an iSO metric screw thread (M2, M4, M6x15, etc).<br>
These are simple connection types,<br>
But they can be more complex with unique names<br>

For example:

    "Type" : "Crosspoint"

    Four M6 screws in a cross formation 10cm from the center of each screw.<br>
    Then give it a unique name, like "M6-cross-10".

Contributor guide:

To contribute a new connection type it must contain:

    - A unique name
    - General and technical description (what it looks like and how it works)
    - One technical drawing
    - One example use
    - Common / intended uses

A contribution must contain a dedication to the public domain.
